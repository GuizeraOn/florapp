# Flora App

Flora App es tu compañera de salud femenina con diagnóstico IA, rituales y herramientas diseñadas especialmente para la mujer.

## Características Principales
- **Diagnóstico IA**: Evaluación personalizada de tu bienestar.
- **Rituales**: Prácticas diarias para mejorar tu salud y equilibrio.
- **Herramientas**: Seguimiento y recursos para tu cuidado personal.

## Ejecución Local

**Requisitos:** Node.js

1. Instala las dependencias:
   `npm install`
2. Configura tu API Key de Gemini en `.env.local` (si es necesario).
3. Inicia la aplicación:
   `npm run dev`
